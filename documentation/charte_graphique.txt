CHARTE GRAPHIQUE - SITE DE SUIVI DE NAVIRES
=========================================

1. PALETTE DE COULEURS
---------------------

Couleurs principales :
- Bleu marine : #1a237e (en-têtes, éléments importants)
- Bleu clair : #1976d2 (boutons, appels à l'action)
- Bleu très clair : #e3f2fd (fonds, surbrillances)

Niveaux de gris :
- Noir : #212121 (texte principal)
- Gris foncé : #757575 (texte secondaire)
- Gris clair : #e0e0e0 (bordures, séparateurs)
- Blanc cassé : #f5f5f5 (arrière-plans)

2. TYPOGRAPHIE
-------------
Police principale : Roboto (300, 400, 500, 700)

Hiérarchie typographique :
- Titres principaux (h1) : Roboto 700, 2.5rem, #1a237e
- Sous-titres (h2) : Roboto 500, 2rem, #1a237e
- Sous-titres secondaires (h3) : Roboto 500, 1.5rem, #1976d2
- Corps de texte : Roboto 400, 1rem, #212121
- Texte secondaire : Roboto 400, 0.875rem, #757575

3. BOUTONS
----------
Bouton principal :
- Fond : #1976d2
- Texte : blanc
- Bordure : aucune
- Coins arrondis : 4px
- Espacement interne : 10px 20px
- Transition : fond 0.3s

Bouton secondaire :
- Même style que le bouton principal mais avec fond blanc et texte bleu clair

Bouton d'alerte :
- Même style que le bouton principal mais avec fond rouge

4. CARTES ET CONTENEURS
----------------------
Carte standard :
- Fond : blanc
- Coins arrondis : 8px
- Ombre portée légère
- Marge inférieure : 20px

Carte surélevée (au survol) :
- Ombre plus prononcée
- Légère élévation
- Transition fluide

5. FORMULAIRES
--------------
Champs de formulaire :
- Bordure légère
- Coins arrondis
- Remplissage interne confortable
- Transition sur la bordure

Champs au focus :
- Bordure bleue
- Légère ombre portée bleutée

6. NAVIGATION
-------------
Barre de navigation :
- Hauteur : 64px
- Fond : bleu marine
- Texte : blanc
- Ombre portée légère

Liens de navigation :
- Texte blanc
- Espacement confortable
- Effet de survol subtil

7. TABLEAUX
-----------
En-tête de tableau :
- Fond bleu très clair
- Texte bleu marine
- Texte en majuscules

Lignes du tableau :
- Bordures légères
- Alternance de couleurs de fond
- Espacement confortable

8. ICÔNES
---------
- Bibliothèque : Material Icons
- Taille standard : 24px
- Couleur principale : bleu clair
- Couleur secondaire : gris foncé

9. ESPACEMENT
------------
- Petit : 8px
- Moyen : 16px
- Grand : 24px
- Très grand : 32px

10. ANIMATIONS
-------------
- Durée standard : 0.3s
- Courbe d'animation : ease-in-out
- Effets de survol subtils

11. DESIGN RESPONSIVE
--------------------
- Mobile : < 768px
- Tablette : 768px - 1024px
- Desktop : > 1024px
